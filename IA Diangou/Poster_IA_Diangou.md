# IA DIANGOU
## Intelligence Artificielle d'Enseignement et de Réponse aux Questions

---

## RÉSUMÉ

IA Diangou est une plateforme d'intelligence artificielle éducative innovante conçue pour enseigner la langue française et répondre aux questions des apprenants de manière personnalisée et adaptative. Le système combine des technologies d'IA avancées (OpenAI, Hugging Face) avec une architecture web moderne (Flask, React, PostgreSQL) pour offrir une expérience d'apprentissage complète et accessible 24/7.

---

## CONTEXTE

**Problématique** :
- Accès limité à l'éducation de qualité pour l'apprentissage du français
- Manque de ressources pédagogiques personnalisées
- Barrières géographiques et économiques
- Besoin d'apprentissage flexible disponible 24/7

**Innovation** : IA Diangou combine l'expertise pédagogique avec l'IA pour créer une solution accessible, adaptative et complète, fonctionnant en mode autonome (sans API payantes) et en mode avancé.

---

## OBJECTIF

**Objectif principal** : Développer une IA capable d'enseigner le français de manière efficace, personnalisée et accessible.

**Objectifs spécifiques** :
1. Enseignement adaptatif (niveau débutant → avancé)
2. Couverture complète : grammaire, orthographe, vocabulaire, syntaxe, prononciation, conjugaison
3. Accessibilité : mode démo gratuit sans dépendance API
4. Gestion complète : professeurs, parents, apprenants, formations, cours
5. Qualité pédagogique : réponses simples, précises, bienveillantes

---

## MÉTHODOLOGIE

**Architecture** :
- **Backend** : Flask (Python) + PostgreSQL
- **Frontend** : React + TypeScript + Tailwind CSS
- **IA** : Mode démo (52+ sujets) + OpenAI API + Hugging Face API

**Système d'IA** :
- Prompt système détaillé (200+ lignes)
- 52+ conditions de détection de sujets
- Conjugaison automatique (8 verbes irréguliers, 5 temps)
- Adaptation automatique du niveau d'enseignement

**Base de données** :
- 7+ tables relationnelles (users, professors, formations, courses, conversations, etc.)
- Sauvegarde de l'historique des conversations
- Gestion des inscriptions et demandes
